# OSCP

### 時間
24時間（攻撃）＋24時間以内（レポート）

<br>

### 内容
- 3台のスタンドアロンサーバ（計60点）
  - 初期侵入：10点（local.txt）
  - 権限昇格：10点（proof.txt）
- 3台のActive Directory（計40点）
  - マシン1：10点
  - マシン2：10点
  - マシン3：10点
 
<br>

### レポート
- 全ての攻撃に対して詳細なステップ、コマンド、コンソール出力を文書化
- 改良、変更したエクスプロイトコード関する詳細
- 明確なスクリーンショット

<br>

### 禁止事項
- Metasploit Pro、Burp Proなどの有償ツール
- LLM、AIなどのチャットボット
- SQLMapやbrowser_autopwnなどの自動攻撃ツール
- Metasploitの複数回利用（1度までOK）
